h1. スマホ向けチャットサンプル

h2. 依存関係

* node.js npmが必要です
* express, socket.io, sencha touchに依存しています
* codeはサーバーもクライアントもcoffee scriptで書かれています

h2. 使い方

<pre><code style="sh">
git clone git://github.com/teeaki/mobilechat.git
npm install express socket.io
node server.js
# or coffee server.coffee
</code></pre>

http://localhost:4000/
